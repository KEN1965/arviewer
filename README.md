📱 GLB & Photo AR Viewer

3Dモデル（.glb）や写真（.jpg / .png）をその場でAR表示できるWebアプリです。
スマホからアクセスするだけで、カメラ映像の中にオブジェクトを出現させられます。
インストール不要・無料・ブラウザだけで動作します！

🌟 機能概要
機能	内容
🧊 GLBファイル表示	3DモデルをWebブラウザ上でAR体験できます。
🖼 写真AR表示	写真を選択してカメラ空間に出現。タップした位置に配置できます。
📱 スマホ対応	Android / iPhone（iOS 17以降）に対応。
☁️ インストール不要	GitHub PagesなどのHTTPS環境でそのまま使えます。
🚀 使い方

スマホでこのページを開く
（※カメラを使用するため、必ずHTTPS環境でアクセスしてください）

「📂 ファイルを選択」ボタンを押します

以下のいずれかを選択します：

.glb：3DモデルとしてAR表示

.jpg / .png：写真としてAR空間に配置

写真モードでは、カメラ画面をタップすると写真が現実空間に出現します✨

🔐 注意事項

カメラアクセスが必要です。初回は「許可」を選択してください。

iPhoneでARが起動しない場合は、設定 → Safari → 「WebXR」機能を有効にしてください。

GLBファイルのサイズが大きい場合、読み込みに時間がかかることがあります。
